3. API для сервісу управління завданнями (Project Management)



Функціональність:*

Реєстрація/авторизація користувачів

Створення проектів

Додавання учасників до проекту

Створення завдань в проекті

Призначення виконавців завдань

Зміна статусу завдань

Коментування завдань

Відстеження прогресу виконання проекту


Технології:*
FastAPI для API
PostgreSQL для зберігання даних про проекти, завдання, користувачів
Redis для кешування даних та оптимізації швидкості
Loki та Grafana для моніторингу та логування
